# Zintro - Product Overview

## What is Zintro?

Zintro is a smart digital introduction platform that enables businesses to create professional, SEO-friendly digital business card microsites. It's a comprehensive multi-brand microsite builder that replaces traditional business cards with dynamic, interactive web experiences accessible via custom URLs and QR codes.

## Technology Stack

### Frontend
- **Next.js 14** - React framework for production-grade applications
- **React 18** - Modern UI library with hooks and concurrent features
- **TypeScript** - Type-safe JavaScript for robust code
- **Tailwind CSS** - Utility-first CSS framework for rapid styling

### Backend
- **Node.js** - JavaScript runtime for server-side logic
- **Prisma ORM** - Type-safe database access layer
- **Express.js** - Web application framework

### Database & Storage
- **PostgreSQL** - Relational database for data persistence
- **Redis** - In-memory cache for performance optimization
- **AWS S3** - Cloud storage for media files
- **CloudFront CDN** - Content delivery network for fast asset loading

### Integrations
- **Stripe & Razorpay** - Payment processing for subscriptions
- **Twilio** - SMS notifications and alerts
- **Nodemailer** - Email delivery system
- **Calendly** - Appointment booking integration
- **Tawk.to, Intercom, Crisp** - Live chat widget support

### Security & Monitoring
- **Speakeasy** - Two-factor authentication (2FA/MFA)
- **JWT** - Secure token-based authentication
- **Sentry** - Error tracking and monitoring
- **Pino** - High-performance logging
